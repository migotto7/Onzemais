# Onzemais

#### Backend dev deploy

- Run mysql

  - `docker run --name onzemais -e MYSQL_ROOT_PASSWORD=my-secret-pw -p 3306:3306 -d mysql:latest`

- Set .env info following the .env.ex

- Install Node

- Run npm i

- npx prisma generate

- npx prisma db push

- npm run start:dev

### Frontend

- npm i

- npm start
